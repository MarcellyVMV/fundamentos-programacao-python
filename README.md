## Sobre a disciplina

A disciplina teve como foco o desenvolvimento do raciocínio lógico-computacional, da resolução algorítmica de problemas e da aplicação de boas práticas de programação utilizando Python.

Ao longo do semestre, foram explorados conceitos fundamentais para a modelagem de problemas, implementação de soluções computacionais e organização de código de forma clara, reutilizável e modular.

### Ementa

- Representação computacional de problemas;
- Desenvolvimento de soluções algorítmicas;
- Aspectos sintáticos e semânticos de linguagens de programação de alto nível;
- Comandos, estruturas de dados e estruturas básicas para construção de soluções computacionais;
- Modelagem e implementação de código modularizado;
- Organização, reutilização e legibilidade de código;
- Princípios de boas práticas de programação.

### Conteúdos abordados

Durante a disciplina, foram estudados tópicos como:

- Lógica de programação
- Variáveis e tipos de dados
- Entrada e saída de dados
- Operadores aritméticos, relacionais e lógicos
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Modularização
- Listas, tuplas e dicionários
- Manipulação de strings
- Manipulação de arquivos
- Boas práticas de desenvolvimento

## Aprendizados desenvolvidos

Ao longo da disciplina, foram desenvolvidas competências como:

- Pensamento lógico e algorítmico;
- Decomposição e modelagem de problemas;

- Depuração e correção de erros;
